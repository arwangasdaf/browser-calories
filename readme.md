# Browser Calories for Chrome

> A Chrome extension to measure your performance budget.

## Setup

1) Install dependencies:

```
npm install
```

2) Transpile scripts:

```
npm start
```

3) Go to `chrome://extensions`

4) Click on `Load unpacked extension...`

5) Select the `dist` folder
